# Image Organizer

## How to install

1. `git clone https://github.com/ozakione/image-organizer`
2. `cd image-organizer`
3. `npm i`
4. `mkdir img`
5. Place all your images in the `img` folder

## How to run

- `node index.js`
- Run `node index.js -h` to display program usage
